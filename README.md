# How to run

**Requirements** : 

1. [dasm](https://dasm-assembler.github.io/)
2. [stella](https://stella-emu.github.io/)
3. make (to make cleaner install for now)

To compile the game just enter into the directory and 

```
$ make all
$ make run
```
